# USER_MANUAL.md — X 團購文產生器 使用手冊

## 概覽

X 團購文產生器是一套本機運行的 AI 輔助工具，將英文、日文或簡體中文的產品說明自動翻譯，並產生適合 X（Twitter）平台的台灣風格團購貼文。適用於 X 付費帳戶，產出單篇完整貼文（無字數限制）。

---

## 工具一覽

| 功能 | 說明 |
|------|------|
| 自動翻譯 | 英文 / 日文 / 簡體中文 → 繁體中文（台灣用語） |
| AI 撰文 | 依台灣團購文風格產出完整貼文 |
| 運費說明 | 國際運費/稅金 checkbox + 物流方式單選（全家 $68/$72/$78 或郵寄），自動附加到貼文 |
| 圖片上傳 | 最多 4 張，拖放支援 |
| 貼文編輯 | 右欄可直接修改 AI 產出內容 |
| 一鍵複製 | 純文字格式，直接貼到 X |
| 歷史記錄 | 儲存草稿，右側抽屜管理 |
| 風格自訂 | 修改 `backend/groupbuy.md` 即時生效 |

## 操作流程

### 步驟一：上傳產品圖片（選填）

- 將圖片拖放至上傳區，或點擊選擇檔案
- 支援 JPG / PNG / WebP
- 最多 4 張；AI 生成時會在貼文中標註圖片位置（📸 見圖1）

### 步驟二：貼入產品說明

選擇來源語言後，貼入原文說明：

| 語言選項 | 適用情境 |
|----------|----------|
| English | 英文原廠描述 |
| 日本語 | 日本官網、樂天說明 |
| 簡體中文 | 淘寶、1688 商品說明 |
| 自動偵測 | 不確定語言時使用 |

### 步驟三：填寫補充資訊（選填）

| 欄位 | 說明 | 範例 |
|------|------|------|
| 產品名稱 | 用於生成貼文標題 | `Sony WH-1000XM5` |
| 原價 | 標示節省金額 | `¥49,500` |
| 團購價 | 明確標示於貼文 | `NT$9,800` |
| 開團日期 | — | `2026-03-20` |
| 截團日期 | — | `2026-03-27` |
| 下單方式 | 告知買家如何下單 | `私訊下單 / Google 表單` |

### 步驟四：設定運費選項

**費用含括（checkbox）**：勾選代表總價已含，未勾選者 AI 會自動以 ⚠️ 標注「不含」。

| 選項 | 說明 | 已含標示 | 未含標示 |
|------|------|----------|----------|
| 國際運費 | 海外寄出的運費 | `✅ 含國際運費` | `⚠️ 不含國際運費` |
| 進口稅金 | 海關稅費 | `✅ 含稅` | `⚠️ 不含稅金` |

**物流方式（單選，三擇一）**：全家店到店與郵寄互斥，只能擇一。

| 選項 | 說明 |
|------|------|
| ⚠️ 不含物流費 | 不包含任何物流，貼文中以 ⚠️ 標示 |
| ✅ 全家店到店 | 已含全家店到店運費，選後再選費率 $68 / $72 / $78 |
| ✅ 郵寄 | 已含郵寄費用，貼文中標示物流方式 |

**運費備註**：可補充說明，如「滿 $2000 免運」、「離島加收 $100」。

### 步驟五：生成貼文

點擊「🚀 翻譯並生成團購文」，系統依序執行：

1. 翻譯原文 → 繁體中文（右欄上方顯示，可修改）
2. AI 依 `groupbuy.md` 風格撰寫完整貼文（右欄下方顯示，可直接編輯）

貼文結構：
```
【吸睛開場】產品名稱 + 價值主張 + 團購價
───────────────────────────
核心賣點（3–5 點 emoji 開頭）
使用體驗（真實感描述）
規格補充（簡潔條列）
運費說明（✅ 已含 / ⚠️ 不含）
團購資訊 + CTA + Hashtag
```

### 步驟六：複製與發文

- 點擊「📋 複製貼文」複製純文字到剪貼簿
- 直接貼到 X 發文（付費帳戶，無字數限制）

### 步驟七：重新生成

修改翻譯結果或補充資訊後，點擊「🔄 重新生成」可重新撰文，**不重新翻譯**，節省 API 呼叫。

### 步驟八：儲存記錄

點擊「💾 儲存記錄」存入 SQLite，可在右上角「📋 歷史記錄」查閱與刪除。

## 自訂撰文風格

編輯 `backend/groupbuy.md`（不需修改程式碼）：

- 可調整語氣（更正式 / 更活潑）
- 可針對特定品類優化（美妝 / 3C / 食品）
- 修改後**立即生效**，無需重啟服務
